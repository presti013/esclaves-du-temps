Statistiques du site

Le site est hébergé sur GitHub Pages (statique) : la mesure d'audience passe par GoatCounter, respectueux de la vie privée (sans cookies, conforme RGPD). Le script de comptage est injecté sur toutes les pages (racine, /en, /it). Cette page n'est qu'un raccourci vers le vrai tableau de bord, qui reste privé chez GoatCounter.

Ouvrir le tableau de bord GoatCounter → Compte / réglages

⚠ Remplacer presti013 par le code de site fourni par GoatCounter (dans cette page ET dans i18n_build.py), puis relancer le build.
